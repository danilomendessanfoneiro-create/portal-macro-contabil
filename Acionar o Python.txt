2️⃣ Vá até a pasta do script

Seu script precisa estar salvo em um arquivo .py.

Exemplo:

C:\oneflow_bi\extrator_oneflow.py

Então no CMD digite:

cd C:\oneflow_bi

Agora o CMD deve ficar assim:

C:\oneflow_bi>


ACionar o programa

PS C:\Users\macro> cd C:\oneflow_bi\
PS C:\oneflow_bi> streamlit run portal.py


Renovar o portal.py

git add portal.py

git commit -m "Atualização Geral"

git push



Fluxo correto sempre que alterar o portal
1️⃣ editar portal.py
2️⃣ salvar arquivo
3️⃣ git add portal.py
4️⃣ git commit -m "mensagem"
5️⃣ git push

Depois disso o Streamlit Cloud / Vercel / servidor atualiza automaticamente.

⏱ Normalmente leva 30s até 2 minutos.
